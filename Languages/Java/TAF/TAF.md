# TAF基础

## TAF开发

引入maven依赖

```xml
<dependencies>
		<dependency>
			<groupId>qq-central</groupId>
			<artifactId>taf-server</artifactId>
			<version>3.0.0-SNAPSHOT</version>
		</dependency>
		<dependency>
			<groupId>qq-central</groupId>
			<artifactId>taf-proxy</artifactId>
			<version>3.0.0-SNAPSHOT</version>
		</dependency>
</dependencies>
```
