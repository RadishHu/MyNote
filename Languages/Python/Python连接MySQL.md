# Python连接MySQL

这里使用pymysql模块连接mysql，在使用pymysql之前需要先安装模块，可以通过pip安装，也可以离线安装

## Linux安装pymysql

- 下载[pymysql](https://pypi.org/)，并上传到服务器

- 解压缩

  ```shell
  tar zxvf PyMySQL-0.9.3.tar.gz
  ```

- 进入解压目录

  ```shell
  cd PyMySQL-0.9.3
  ```

- 安装pymysql(需要root权限)

  ```shell
  sudo python setup.py install
  ```
