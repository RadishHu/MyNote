## 文件/目录方法

OS模块提供的处理文件和目录，常用的方法有：

**os.access(path,mode)**

检验权限模式

**os.chdir(path)**

改变当前工作目录

**os.chflags(path,flags)**

设置路径的标记为数字标记

**os.chmod(path,mode)**

更改权限

**os.chown(path,uid,gid)**

更改文件所有者

**os.chroot(path)**

更改当前进程的根目录

**os.close(fd)**

改变当前进程的根目录
